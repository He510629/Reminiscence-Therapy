# 搪瓷缸子内容设计

## 目标

把现有“搪瓷缸子”升级成一条完整的策展内容，并把目前已经完成重点打磨的几条内容固定排在前面展示，方便老人进入策展馆后先看到最适合回忆和停留的内容。

## 内容方向

“搪瓷缸子”不按器物知识做说明，而按“家家户户都用过的一只缸子”来讲述。正文围绕喝水、上班、出门、串门和过日子的细节展开，让老人先从手感、图案、热气和旧习惯里进入记忆。

正文按四段推进：

1. 早晨泡茶：讲搪瓷缸子放在桌角、倒开水、冒热气的熟悉画面。
2. 单位上班：讲上班带缸子、茶叶浮着、桌边一放就是半天的旧日节奏。
3. 串门待客：讲家里来人时递上一只搪瓷缸子的朴素热情。
4. 留在柜里：讲后来杯子越来越多，但老人还是会记得那只最常用的缸子。

## 图片策略

- 使用 1 张封面图和 4 张画廊图。
- 图片尽量选真实搪瓷缸、带字样的旧式搪瓷杯和博物馆藏品。
- 来源优先 Wikimedia Commons，保证引用稳定。

## 排序策略

把以下内容固定排在前面：

1. 钟楼旧影
2. 回民街的泡馍馆
3. 老火车站
4. 城墙根下的早市
5. 搪瓷缸子

推荐页与“全部内容”页都按这组优先顺序展示，其余内容保持原有顺序跟在后面。

## 实现方式

- 继续沿用 `content.py` 中“特殊标题注入增强内容”的方式。
- 在 `main.py` 启动种子逻辑里同步“搪瓷缸子”的封面、摘要和标签。
- 在内容服务层增加统一排序函数，让推荐和列表都能复用。
